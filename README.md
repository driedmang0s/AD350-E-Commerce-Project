#### E-Commerce Database Project

##### Project Overview
This project is an e-commerce marketplace database. The goal was to design a database, build it in MySQL, add sample data, run queries to obtain data analytical information, and create a Tableau dashboard for visualization. 

##### What the Database tracks:
  - Users
  - Products
  - Inventory
  - Transactions
  - Transaction items
  - Reviews

##### Tables used:
  - User: Stores customer information
  - Product: Stores product information
  - Inventory: Tracks product stock
  - Transaction: Stores orders
  - Transaction_Item: Stores products inside each order
  - Review: Stores product reviews

##### Dashboard:
The Tableau Dashboard shows:
  - Current inventory
  - Low stock products
  - Most popular products
  - Least popular products
  - Purchases by Inactive users
  - Newest products
  - Best performing categories

##### Tools Used:
  - MySQL Workbench
  - MySQL
  - Tableau Public 2026.1
  - Powerpoint
  - Github
  - LucidChart (Database ERD)

##### How to Open the Tableau Workbook:
  1) Download or clone this GitHub repository.
  2) Open Tableau Public or Tableau Desktop (whicchever one you have currently).
  3) Open the Tableau workbook file.
  4) If Tableau asks for data files, reconnect the .csv files from the Dashboard folder.
  5) Go to the 'Simple E-Commerce Dashboard' tab to view the final dashboard.
  6) Use the filters in the dashboard to explore the data visualization. Enjoy!
